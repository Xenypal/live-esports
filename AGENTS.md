# Repo Instructions

- Use the local skill at `.codex/skills/playwright-functional-testing` whenever you create or change browser-visible functionality in this repo.
- Add or update at least one relevant Playwright test for browser-visible behavior changes unless the change is purely non-UI.
- Run the narrowest relevant Playwright coverage before finishing, and report what command you ran plus any remaining gaps.
- Prefer mocking LoL Esports and Data Dragon API responses in Playwright tests instead of depending on live external data.
